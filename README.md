#University Work

これは学校の授業で作成したC言語をまとめたレポジトリです。

##内容
- kadai12.c:一回生後期の作品、学習して数か月の作品。数当てゲームプログラムで、以下の機能を持つ：
  - 3ターンの数当てゲーム（1-100のランダムな数）
  - 各ターンの予想回数を記録し、平均でスコア計算
  - ランキングシステム（トップ10、メモリ上のみ）
  - 名前入力（ローマ字、数字、ピリオドのみ）
  - 入力検証（数字入力チェック）
  - 同じスコアの場合の順位表記調整
- kadai13.c:一回生後期の作品、学習して数か月の作品。数当てゲームプログラムで、以下の機能を持つ：
  - 3ターンの数当てゲーム（1-100のランダムな数）
  - 各ターンの予想回数を記録し、平均でスコア計算
  - ランキングシステム（トップ10保存）
  - 名前入力（ローマ字、数字、ピリオドのみ）
  - ランキングの保存・読み込み（ファイル使用）
  - 同じスコアの場合の順位表記調整
- shell.c: シンプルなシェルプログラムの実装。C言語で書かれ、以下の機能を持つ：
  - コマンドライン引数の解析と実行
  - パイプ機能の実装
  - バックグラウンドジョブの管理（最大100ジョブ）
  - シグナルハンドリング（SIGCHLD、SIGINT）
  - PATH環境変数からの実行ファイル検索
- report1.c: JR西日本の運輸収入データを管理するプログラム。ダブルリンクリスト構造を使用し、以下の機能を持つ：
  - データ構造（年月、取扱収入対前期比、利用客内訳、山陽新幹線利用状況）
  - 一覧表示機能（全データ、特定項目のみ）
  - 正順・逆順表示
