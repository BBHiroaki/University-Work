# University Work

このリポジトリは、大学の授業で作成したC言語の課題プログラムをまとめたものです。主に数当てゲーム、簡易シェル、データ管理プログラムに関する実装が含まれています。

## 収録ファイル

- `kadai12.c`
  - 一回生後期の課題
  - 3ラウンドの数当てゲーム（1〜100）
  - 各ラウンドの推測回数を記録し、平均値でスコアを計算
  - 順位表（トップ10）をメモリ上で管理
  - 名前入力の検証（ローマ字、数字、ピリオドのみ許可）
  - 数字入力のバリデーション
  - 同スコア時の順位表記の調整

- `kadai13.c`
  - 一回生後期の改良版
  - `kadai12.c`と同様の数当てゲーム機能
  - ランキング情報をファイルに保存・読み込み
  - 永続的なスコア管理（`ranking.dat`を使用）

- `shell.c`
  - 二回生後期の課題
  - 簡易シェルの実装
  - 外部コマンドの実行と引数解析
  - パイプ(`|`)によるコマンド連結
  - 出力リダイレクト(`>`)対応
  - バックグラウンド実行(`&`)とジョブ管理（最大100ジョブ）
  - `setenv`, `unsetenv`, `jobs`, `fg` のビルトインコマンド
  - `SIGCHLD` と `SIGINT` のシグナル処理
  - `PATH` 環境変数から実行ファイルを探索

- `report1.c`
  - 二回生前期の課題
  - JR西日本の運輸収入データを扱う管理プログラム
  - 2重リンクリスト（双方向リスト）を利用したデータ構造
  - 全件表示、特定項目表示、正順・逆順表示
  - 年月検索や各種データ検索機能
  - 各項目ごとのソート処理（バブルソート、選択ソート、挿入ソート）

- `ranking.dat`
  - `kadai13.c` のランキングデータ保存用ファイル

## ビルド方法

以下のコマンドでそれぞれコンパイルできます。

```bash
gcc -o kadai12 kadai12.c
gcc -o kadai13 kadai13.c
gcc -o shell shell.c
gcc -o report1 report1.c
```

## 実行例

- 数当てゲーム

```bash
./kadai12
./kadai13
```

- シェルプログラム

```bash
./shell
```

- 収入データ管理

```bash
./report1
```

## 補足

- すべてのプログラムはC言語で実装されています。
- Linux環境での動作を想定しています。
- `kadai13.c` は `ranking.dat` を読み書きするため、このファイルが存在しない場合も新規作成されます。
