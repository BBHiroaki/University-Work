#University Work

これは学校の授業で作成したC言語をまとめたレポジトリです。

##内容
- kadai12.c:一回生後期の作品、学習して数か月の作品。数当てゲームプログラムで、以下の機能を持つ：
  - 3ターンの数当てゲーム（1-100のランダムな数）
  - 各ターンの予想回数を記録し、平均でスコア計算
  - ランキングシステム（トップ10、メモリ上のみ）
  - 名前入力（ローマ字、数字、ピリオドのみ）
  - 入力検証（数字入力チェック）
  - 同じスコアの場合の順位表記調整
- kadai13.c:一回生後期の作品、学習して数か月の作品。数当てゲームプログラムで、以下の機能を持つ：
  - 3ターンの数当てゲーム（1-100のランダムな数）
  - 各ターンの予想回数を記録し、平均でスコア計算
  - ランキングシステム（トップ10保存）
  - 名前入力（ローマ字、数字、ピリオドのみ）
  - ランキングの保存・読み込み（ファイル使用）
  - 同じスコアの場合の順位表記調整
